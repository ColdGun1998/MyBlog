# Git基本操作

Git 常用的是以下 6 个命令：git clone、git push、git add 、git commit、git checkout、git pull，后面我们会详细介绍。

# 创建版本库

git init 将当前的空目录初始化为Git可以管理的仓库
git add [fileName] 添加一个或多个文件到暂存区
git commit -m "提交说明" 将暂存区内容添加到本地仓库中

其他

git add -A  提交所有变化
git log 显示从最近到最远的提交日志
git reset [--soft | --mixed | --hard] [HEAD] 用于回退版本，可以指定退回某一次提交的版本
git reset HEAD^            # 回退所有内容到上一个版本  
git reset HEAD^ hello.php  # 回退 hello.php 文件的版本到上一个版本  
git  reset  052e           # 回退到指定版本

# 创建与合并分支

git checkout -b dev 创建dev分支，然后切换到dev分支，等同于以下两条命令
git branch dev
git checkout dev
git merge dev 用于合并指定分支到当前分支

# 远程仓库

git remote add [shortname] [url] 添加一个新的远程仓库，并指定一个简单的名字
git remote add origin git@github.com:ColdGun1998/MyBlog.git
git push origin masterlocal:masterOrigin 将本地的 masterlocal 分支推送到 origin 主机的 masterOrigin 分支
git push origin master 和上一条命令等同，如果本地分支名与远程分支名相同，则可以省略冒号
git push --force origin master 如果本地版本与远程版本有差异，但又要强制推送可以使用 --force 参数：
